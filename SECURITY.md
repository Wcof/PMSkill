# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch state.

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue first.

Please report privately to project maintainers with:

- Affected component/path
- Reproduction steps
- Potential impact
- Suggested mitigation (if available)

We will acknowledge receipt and provide an initial triage response as soon as possible.
