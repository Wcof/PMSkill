# Relate 模块指南

## 做什么

建立需求、页面、功能、规则、数据、验收之间的关系链路。

## 输入

- `docs/prd-helper/02-refine/` 下的精炼结果

## 输出

输出到 `docs/prd-helper/03-relate/`：

- `page-map.md`
- `feature-map.md`
- `rule-map.md`
- `data-map.md`
- `acceptance-map.md`
- `context-map.md`
- `check.md`

## 验收条件

- 每个 fact 关联到页面或功能
- 每个 feature 关联到规则
- 每个 rule 关联到数据对象
- 每个页面关联验收标准
- 无孤立需求、孤立页面、孤立规则
- 输出 `check.md`
