# Agent 上下文：前端开发

## 1. 本上下文用途

为前端开发 Agent 提供巡检点管理功能的开发上下文，确保 Agent 理解页面结构、字段含义和交互规则。

开发巡检点管理页面，包括列表展示、新建/编辑表单、详情查看、关联机器人点位功能。

## 3. 必须遵守的事实

- 巡检点和机器人点位是不同对象（fact_001）
- 巡检点必须关联到区域（constraint_001）
- 机器人点位必须关联到巡检点（constraint_002）

## 4. 不能擅自推断的内容

- 不能假设巡检点命名规则已统一（question_001 待确认）
- 不能假设检测线的具体定义（question_004 待确认）

- page_004：点位管理（机器人点位）

## 6. 相关功能规则

## 8. 验收标准
