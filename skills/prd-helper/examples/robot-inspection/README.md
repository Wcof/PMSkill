# Robot Inspection Example

## 使用场景

机器人巡检场景用于演示 PRD Helper Skill 如何把产品口述、会议纪要、原型说明、客户反馈和 Agent 对话摘要转成完整 PRD 上下文。

## 输入材料

- 产品口述：地图、区域、巡检点、机器人点位、采集位、设施设备、设施部件、任务计划。
- 会议纪要：巡检点与机器人点位拆分、设施设备层级、检测线定义、任务中断恢复。
- 原型说明：地图管理、点位管理、设施设备管理、任务创建、巡检计划。
- Agent 对话摘要：曾混淆巡检点和机器人点位、机器人和设施设备、任务派生链路。

## 预期输出

- `01-collect/`：保留原始材料和采集检查。
- `02-refine/`：提取事实、背景、目标、决策、约束、冲突、问题、推断。
- `03-relate/`：建立页面、功能、规则、数据、验收的关联。
- `04-generate/`：生成项目说明、页面说明、功能规则、数据说明、验收标准和 Agent 上下文。
- `05-check/`：输出最终检查、来源检查、关系检查、生成检查和 Context Delta。

完整 demo 位于 `demo/robot-inspection-demo/docs/prd-helper/`。
