# Discovery

需求发现——从模糊想法到清晰的 PMContext。

## User-invoked

- **[pm-need](./pm-need/SKILL.md)** — 从模糊想法或用户诉求出发，收集材料并追问澄清，沉淀成 PMContext。

## Model-invoked

- **[pm-collect](./pm-collect/SKILL.md)** — 收集零散想法、用户反馈、会议纪要、历史会话，整理到 collect 目录。
- **[pm-refine](./pm-refine/SKILL.md)** — 对已收集材料追问澄清，区分事实/假设/冲突/待确认，沉淀成 PMContext。
