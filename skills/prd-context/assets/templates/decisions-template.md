## 填写说明

- **decision_id**：唯一标识，格式 `decision-XXX`
- **决策内容**：简洁描述设计决策
- **决策原因**：为什么做出这个决策
- **来源**：引用原始材料文件路径
- **影响范围**：该决策影响的模块或功能
- **状态**：已确认 / 待确认 / 已否决 / 已过时
