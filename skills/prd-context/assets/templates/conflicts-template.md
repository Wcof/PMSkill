# 冲突点

> 原始材料中发现的矛盾、不一致或需要裁决的冲突。

## 冲突清单

| conflict_id | 冲突描述 | 冲突来源 | 待决策问题 | 需要确认的问题 | 影响范围 | 处理状态 |
|-------------|---------|---------|-----------|--------------|---------|---------|
| conflict-001 | | | | | | 待处理/已解决/已搁置 |

## 填写说明

- **conflict_id**：唯一标识，格式 `conflict-XXX`
- **冲突描述**：描述冲突的具体内容
- **冲突来源**：冲突涉及的两方或多方来源
- **待决策问题**：需要裁决的具体问题
- **需要确认的问题**：为解决冲突需要进一步确认的问题
- **影响范围**：该冲突影响的模块或功能
- **处理状态**：待处理 / 已解决 / 已搁置
