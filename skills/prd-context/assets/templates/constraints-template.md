## 填写说明

- **constraint_id**：唯一标识，格式 `constraint-XXX`
- **约束内容**：简洁描述约束条件
- **约束类型**：业务约束 / 技术约束 / 合规约束 / 资源约束
- **来源**：引用原始材料文件路径
- **影响页面**：该约束影响的页面
- **影响规则**：该约束影响的业务规则
- **状态**：已确认 / 待确认
