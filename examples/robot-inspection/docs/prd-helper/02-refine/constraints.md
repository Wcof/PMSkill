# 业务约束清单

## constraint_001：巡检点必须关联到区域

- 约束内容：每个巡检点必须属于一个区域，不能独立存在于地图之外
- 约束类型：业务
- 影响对象：巡检点管理
- 违反后果：巡检点无法在地图上定位
- 来源材料：产品口述
- 来源位置：01-collect/product-voice.md
- 状态：已确认

## constraint_002：机器人点位必须关联到巡检点

- 约束内容：每个机器人点位必须属于一个巡检点，不能独立存在
- 约束类型：业务
- 影响对象：点位管理
- 违反后果：机器人点位无法关联到业务巡检单元
- 来源材料：产品口述、会议纪要
- 来源位置：01-collect/product-voice.md；01-collect/meeting-notes.md
- 状态：已确认

## constraint_003：设施部件必须关联到设施设备

- 约束内容：每个设施部件必须属于一个设施设备，不能独立存在
- 约束类型：业务
- 影响对象：设施设备管理
- 违反后果：部件无法追溯到所属设备
- 来源材料：会议纪要
- 来源位置：01-collect/meeting-notes.md
- 状态：已确认

## constraint_004：任务必须有执行机器人

- 约束内容：每个任务必须指定执行的机器人，不能创建无机器人的任务
- 约束类型：业务
- 影响对象：任务管理
- 违反后果：任务无法执行
- 来源材料：原型说明
- 来源位置：01-collect/prototype-notes.md
- 状态：已确认

## constraint_005：检测结果必须关联到设施设备或部件

- 约束内容：检测结果必须记录到对应的设施设备或部件上，不能独立存在
- 约束类型：数据
- 影响对象：巡检管理、设施设备管理
- 违反后果：检测结果无法追溯
- 来源材料：会议纪要
- 来源位置：01-collect/meeting-notes.md
- 状态：已确认

## constraint_006：地图是区域的容器

- 约束内容：区域必须属于地图，不能独立存在
- 约束类型：业务
- 影响对象：地图管理
- 违反后果：区域无法在地图上展示
- 来源材料：产品口述
- 来源位置：01-collect/product-voice.md
- 状态：已确认
