# 页面说明：巡检点管理

## 1. 页面基本信息

- 页面路径：/inspection-point/list
- 所属模块：巡检管理
- 使用角色：产品经理、实施工程师、巡检人员
- 页面目标：管理巡检点，支持创建、编辑、删除巡检点，关联机器人点位
- 来源事实：fact_001、fact_009、fact_010
- 关联规则：rule_003
- 关联数据：data_003

## 2. 页面区域

| 区域 | 用途 | 内容 |
|---|---|---|
| 搜索区 | 按条件筛选巡检点 | 关键词搜索、区域筛选、状态筛选 |
| 列表区 | 展示巡检点列表 | 名称、编号、所属区域、关联点位数、状态、操作 |
| 操作区 | 执行操作 | 新建、编辑、删除、批量操作 |

## 3. 字段说明

| 字段 | 含义 | 来源 | 状态 |
|---|---|---|---|
| name | 巡检点名称 | 用户输入 | 已确认 |
| code | 巡检点编号 | 系统生成或用户输入 | 已确认 |
| zone_id | 所属区域 | 关联选择 | 已确认 |
| robot_points | 关联机器人点位 | 关联选择 | 已确认 |
| status | 状态 | 系统管理 | 已确认 |

## 4. 用户操作

| 操作 | 入口 | 结果 | 关联规则 |
|---|---|---|---|
| 新建巡检点 | 点击"新建"按钮 | 打开新建表单 | rule_003 |
| 编辑巡检点 | 点击行"编辑"按钮 | 打开编辑表单 | rule_003 |
| 删除巡检点 | 点击行"删除"按钮 | 确认后删除 | rule_003 |
| 查看详情 | 点击行"详情"按钮 | 跳转详情页 | - |
| 关联点位 | 详情页"关联点位" | 选择机器人点位 | rule_004 |

## 5. 页面状态

| 状态 | 场景 | 页面表现 |
|---|---|---|
| 正常 | 数据正常加载 | 展示巡检点列表 |
| 空状态 | 无巡检点数据 | 展示空状态提示和新建引导 |
| 加载中 | 数据加载中 | 展示加载动画 |

## 6. 异常状态

- 网络异常：展示错误提示，支持重试
- 删除失败：展示失败原因（如存在关联数据）

## 7. 权限展示

- 无权限：隐藏新建、编辑、删除按钮
- 只读权限：只展示列表，不展示操作按钮

## 8. 前端验收标准

- 列表支持分页、排序
- 搜索支持关键词和区域筛选
- 新建/编辑表单有必填校验
- 删除有二次确认
- 关联点位支持多选

## 9. 待确认问题

- question_001：巡检点命名规则是否需要统一
- question_004：检测线的具体定义

本页面说明基于：产品口述、原型说明、客户反馈。
